Welcome to CS198-75! For your first homework, you'll be building out a single-page complete personal portfolio that can serve as your own site for showcasing your education, work experience, achievements, and projects using HTML and CSS. To fetch the skeleton, you will git clone the skeleton repository into your cubstart directory. While at your cubstart directory, run the following bash command:

Part 1: The Document Head

Access the index.html file in the project directory. This is where the layout of your site is; think of it as the skeleton of the site.

Recall that the head of the document is where metadata, page information, and important link tags go. You are able to link style sheets (your .css files) within the document head, as well as title your page as it shows on your browser tab.

Add a title for your website, and link your stylesheet in the enclosing head tags. The corresponding tags are referenced for your convenience.

<link rel="" href=""><title>Your Name</title>

Part 2: HTML Tags and Attributes

Here, you'll be creating your logo, which in this example will be your initials. Within the h3 tags in the problem, you will add an a tag with your initials, with the class of "link" linking to the same page that it is on (referencing the current path is possible with "/").

Part 3: CSS Flex Layout

Take a look at the header tag, with classes "header" and "center." This entire section is your page's header, which includes your logo, navigation, and other buttons. Think about how a header like this is laid out across the top of a page. What kinds of orientation would you need for its elements?

<header class="header center">

Within the styles.css file, you'll find a class selector for the class center. The intent of this class is to vertically center the children of elements of this class using a flex-box layout. Accomplish this task by adding the two necessary styles to achieve this.

Underneath, there is also a class selector for the class header. Because our header in index.html includes both classes "header" and "center," styling the "header" class will also cause stylistic changes on our header element. Recall that justify-content is the attribute that pertains to manipulating elements in the horizontal direction in a flex-box context. Choose the style for justify-content that you feel would best fit the look of the header tag and its children.

Part 4: Basic Animation with CSS

You will now create a simple underline animation with CSS. You will notice that the styles in this section have a strange selector:

.link::before {
    content: "";
    display: inline;
    width: 0%;
    height: 0.2em;
    position: absolute;
    bottom: 0;
    background-color: var(--clr-primary);
    /* transition: */
}

The ::before selector is called a "pseudo-selector," which essentially just refers to a pseudo-element that exists before the selected element. You can give this pseudo-element an appearance by setting its display to a specific attribute. In this case, we are using it to create our text underline. You'll notice now, if you try to hover the links on the header, you will see the underline, but it just appears instantly. We want to create this animation, so that it comes out from the left.

The transition property provides a way to control animation speed when changing CSS properties. Instead of having property changes take effect immediately, you can cause the changes in a property to take place over a period of time. Property changes are found in elements that have properties that change on certain actions, for instance hover (mouse over) and focus (selected). The property changes can be defined by :hover and :hover selectors as such: 

.link:hover::before, .link:focus::before {
    width: 100%;
}

Observe that the way the underline appears. When the link is either hovered or selected, the width of the pseudo-element .link::before changes to 100%, else it is 0 when it isn't hovered/selected. We want to animate this change. Fill in the transition style so that we achieve a smooth animation.

transition: [property] [duration] [timing-function];

In the property section, you will want to target width as mentioned previously. As for the timing-function, you can use ease-in, but there are many options available. Try playing around with the timing of the animation and see the results by hovering your links and logo!
